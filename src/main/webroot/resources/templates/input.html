How it works

This tool applies the methods to perform a quantitative risk analysis based on inputs of loss event frequency and loss event magnitude for a single primary and secondary risk. The tool uses the data for a PERT analysis followed by a Monte-Carlo simulation to produce a graph and analysis of the total loss exposure for the risks.
